# 🌅 Sunset ARC

**AI-powered automation platform** - Connect your apps, automate workflows, and let AI handle the busywork.

## Features

- 🤖 **AI Chat Interface** - Natural language commands to automate tasks
- 📧 **Email Integration** - Send emails via Gmail
- 📅 **Calendar Integration** - Manage Google Calendar events
- 💬 **Slack Integration** - Post messages to Slack channels
- 📊 **Meta Ads** - Manage Facebook ad campaigns
- 🎨 **Canva** - Create designs programmatically
- 💾 **Persistent Memory** - Conversation history saved per project

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open http://localhost:3000 in your browser.

## Tech Stack

- **Frontend**: React + Vite
- **Backend**: Supabase Edge Functions
- **AI**: Claude (Anthropic)
- **Integrations**: Composio
